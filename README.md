* 本程式開發純屬作者興趣，非商業用途

* 本程式參考 [易學網](https://www.eee-learning.com/article/2076) 及 [Wiki 周易六十四卦列表](https://zh.wikipedia.org/wiki/%E5%91%A8%E6%98%93%E5%85%AD%E5%8D%81%E5%9B%9B%E5%8D%A6%E5%88%97%E8%A1%A8)
